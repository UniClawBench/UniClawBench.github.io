# Q3 OKRs — Platform Engineering

## Objective 1: Improve developer experience
- KR1: Reduce CI median runtime from 18 minutes to 9 minutes
- KR2: Migrate 80% of repos off the legacy build script
- KR3: Achieve a 4.0/5.0 internal NPS on tooling

## Objective 2: Strengthen production reliability
- KR1: Drive p95 latency on the public API below 250ms
- KR2: Reduce paged incidents by 30% quarter-over-quarter
- KR3: Add end-to-end tracing coverage on the top 10 critical paths

## Objective 3: Hire and onboard the team for FY26
- KR1: Close two senior platform engineer roles
- KR2: Close one staff site-reliability engineer role
- KR3: Each new hire shipping their first review-merged PR within 14 days

## Health metrics (not committed but tracked)
- Test flakiness rate
- Median PR review time
- Open dependency-update PRs
